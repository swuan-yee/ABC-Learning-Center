About us page

Our Program

     This website is aimed for the people who can learn programming and website design. The students can learn by using their internet service and they can learn everywhere and everytime by using this website. We teach programming by using both Java and Python. Our web design includes the types of HTML, CSS, JavaScript and jquery.

     When you have completed programming course with Java, you will get a certificate from ABC Learning Centre and you can work as a Java Programmer so does with Python. When you have completed web design course, you will get a certificate from ABC Learning Centre and you can work as a website developer.

Who can learn...

     People who can use the computer basically can start from the foundation level. If you have completed the foundation level, you can directly join the advanced level. People who have interests in programming and website design can join this program. There is no limitation in age.
